# 像素图片资源

本目录存放游戏的像素图片资源。

## 资源规格
- **玩家角色**: 24x32 像素
- **敌人**: 24x24 像素  
- **金币**: 16x16 像素
- **生命道具**: 16x16 像素
- **平台瓦片**: 16x16 像素

## 文件列表
- player.png - 玩家角色精灵图
- enemy.png - 敌人精灵图
- coin.png - 金币精灵图
- heart.png - 生命道具精灵图
- tile.png - 平台瓦片

## 注意
当前游戏使用 Canvas 直接绘制像素，无需外部图片资源。
如需添加图片，请将 PNG 格式的透明背景图片放入此目录。
