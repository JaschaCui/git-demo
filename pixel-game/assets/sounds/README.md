# 8-bit 音效资源

本目录存放游戏的 8-bit 风格音效文件。

## 音效列表
| 文件名 | 描述 | 触发条件 |
|--------|------|----------|
| jump.wav | 跳跃音效 | 玩家跳跃时 |
| coin.wav | 金币音效 | 收集金币时 |
| hurt.wav | 受伤音效 | 玩家受到伤害时 |
| gameover.wav | 结束音效 | 游戏结束时 |

## 格式要求
- 格式: WAV
- 采样率: 44100Hz
- 位深: 16-bit
- 声道: 单声道

## 注意
当前游戏使用 Web Audio API 程序化生成 8-bit 音效，无需外部音频文件。
如需使用自定义音效，请将 WAV 格式文件放入此目录。
